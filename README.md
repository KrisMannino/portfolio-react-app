# Kris Mannino — Portfolio

Personal portfolio site built with React and Vite, deployed to GitHub Pages at [krismannino.com](https://krismannino.com).

## Tech Stack

- React
- Vite
- Tailwind CSS
- GitHub Pages

## Getting Started

### Install dependencies
```bash
npm install
```

### Run locally
```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### Build for production
```bash
npm run build
```

### Deploy to GitHub Pages
```bash
npm run deploy
```

This builds the app and publishes it to the `gh-pages` branch.
